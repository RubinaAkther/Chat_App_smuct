# 💬 Mastery Flutter Chat App

A modern real-time chat application built with **Flutter**, **Firebase**, and **GetX**.  
This project focuses on clean UI, scalable architecture, and real-time messaging features.

---

### 🛠️ Initial Setup
- Created Flutter project: `mastery_flutter`
- Configured Flutter SDK: `^3.11.1`
- Added required dependencies:
  - firebase_core
  - firebase_auth
  - cloud_firestore
  - get
  - google_fonts
  - uuid

---

### 🔥 Firebase Integration
- Connected Flutter app with Firebase
- Initialized Firebase in `main.dart`
- Configured `firebase_options.dart`
- Ready for authentication and Firestore database operations
